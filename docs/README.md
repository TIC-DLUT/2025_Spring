## 简介

本项目作为 DLUT-TIC 的 2025 年春季例会的实操项目。

本项目作为 ChatBox 的前端，技术栈为 Vue3 + TypeScript + Vite + Element Plus + Pinia。

不管你有没有参加该系列的例会，都可以采用本项目来作为前端入门方面的学习。我们将涉及到：

- 现代前端项目的工程化设计
- 如何对 axios 进行二次封装，为什么要封装
- 如何使用 SSE 实现流式对话
- Markdown 渲染与安全防护
- 状态管理、路由守卫等核心概念

Have fun! 爱来自 TIC

## 学习指南

1. 遇到问题及时咨询学长，提问是一种艺术
2. 建议例会到场，讲解更加清晰，并且可以现场拷打讲例会的同学
3. 下面的学习顺序尽量不要更改，也不要跳读，除非你确实掌握了
4. 该项目完成时间比较匆忙，因此，遇到任何问题都欢迎提 issue，或者发送 pr 帮我们解决相关的问题

## 前置要求

- 了解 HTML、CSS、JavaScript 基础语法
- 了解 HTTP 请求的基本概念（GET/POST、Header、Body）
- 已安装 Node.js（建议 18+）和 npm
- 有一个顺手的编辑器（推荐 VS Code + Volar 插件）
